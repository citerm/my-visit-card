## Критерии приемки

В результате выполнения лабораторной работы у вас должен получиться одностраничный персональный сайт, который включает:

• **Статическую информацию о себе**, оформленную в HTML и CSS (имя, фотография, описание, список интересов и т.д.).

• **Динамически добавляемое содержимое с помощью JavaScript**: например, список увлечений, сформированный из массива, текст приветствия, изменяющийся в зависимости от времени суток, и прочие данные, выводимые из JavaScript-объектов.

• **Интерактивные элементы**: поле ввода имени и кнопка для персонализации приветствия, кнопка для переключения темы (светлая/тёмная), кнопки для удаления или изменения контента на странице, визуальные эффекты при наведении курсора и т.п.

• **Изменение стилей через JS**: применение темной темы через добавление/удаление класса на элементах, динамическое изменение отдельных стилей элементов при взаимодействии (например, скрытие/отображение, изменение прозрачности или цвета).

• **Хранение данных в браузере**: сохранение имени пользователя или предпочтений темы в localStorage и демонстрация использования cookies (например, отображение даты последнего визита).

## Требования

Весь функционал должен быть реализован на чистом JavaScript, без 
привлечения сторонних библиотек. Страница должна корректно работать в 
современном браузере. 

После завершения работы убедитесь, что страница обновляется без ошибок, а 
все указанные функции работают согласно заданию. Вы можете творчески 
подойти к оформлению и содержанию персонального сайта, добавив 
дополнительные элементы или стилизовав его по своему вкусу, однако все 
перечисленные обязательные требования должны быть выполнены.
